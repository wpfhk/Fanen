# 파낸 (Fanen) — Product Requirements Document

> PM Agent Team 종합 분석 결과
> 생성일: 2026-03-24
> 버전: 1.0

---

## Executive Summary

| 관점 | 내용 |
|------|------|
| **Problem** | 1,410만 개인투자자 중 대다수가 기관 대비 정보 비대칭에 놓여 있으며, 기존 HTS/MTS는 전문용어와 복잡한 UI로 진입 장벽이 높다. 특히 50대 이상 투자자 증가 추세에도 이들을 위한 접근성 높은 서비스가 부재하다. |
| **Solution** | AI 코치 "핀이(FinAI)"가 뉴스 영향도 분석, 섹터 인과관계, 배당 시뮬레이션을 쉬운 언어와 음성으로 제공하는 투자 정보 플랫폼. 모의투자 게임으로 학습-실전 전환을 유도한다. |
| **Target User** | 정보 소외 투자자 — 20대 초보(학습 니즈), 40대 직장인(시간 부족), 60대 은퇴자(배당 현금흐름) |
| **Core Value** | "세상이 움직이면, 파낸이 먼저 압니다" — 복잡한 금융 정보를 누구나 이해하고 활용할 수 있도록 민주화 |

## 1. 5-Step Discovery Chain

| # | 기회 영역 | 설명 |
|---|----------|------|
| O1 | 뉴스 기반 투자 인사이트 | 뉴스가 종목/섹터에 미치는 영향을 AI가 실시간 해석하여 "신호등" 형태로 제공 |
| O2 | 음성 기반 AI 코치 | 운전 중, 가사 중에도 음성으로 투자 정보를 묻고 받을 수 있는 AI 코치 |

# Part 2: Strategy Analysis

## 2. Value Proposition — JTBD 6-Part Framework

### 2.1 Job Statement

| Part | 내용 |
|------|------|
| **When** | 뉴스에서 경제 이벤트를 접했을 때 / 매월 투자 포트폴리오를 점검할 때 / 은퇴 후 안정적 수입원을 설계할 때 |
| **I want to** | 그 이벤트가 내 투자에 어떤 영향을 미치는지 쉽게 이해하고, 데이터에 기반한 판단을 내리고 싶다 |
| **So I can** | 손실을 줄이고, 기회를 포착하며, 장기적으로 안정적인 투자 수익을 달성할 수 있다 |
| **Functional Job** | 뉴스/이벤트의 종목 영향도를 신호등으로 확인하고, 배당 시뮬레이션으로 미래 현금흐름을 설계한다 |
| **Emotional Job** | "나도 전문가처럼 시장을 읽을 수 있다"는 자신감을 느끼고, 투자에 대한 불안을 줄인다 |
| **Social Job** | 모의투자 랭킹에서 친구들과 경쟁하며 투자 실력을 증명하고, 투자 경험을 공유한다 |

### 2.2 Gains & Pains

| Pains (해소해야 할 고통) | Gains (제공해야 할 이득) |
|------------------------|------------------------|
| 뉴스를 봐도 투자 판단에 어떻게 적용할지 모름 | AI가 뉴스 → 종목 영향을 "신호등"으로 즉시 변환 |
| HTS/MTS 전문용어가 어려워 화면을 이해 못함 | 쉬운 언어 + 음성으로 시니어도 사용 가능 |
| 배당 투자의 장기 효과를 직관적으로 파악 어려움 | 배당 재투자 복리 효과를 시각적 그래프로 제공 |
| 실전 투자 두려움 (돈을 잃을까봐) | 모의투자로 리스크 없이 연습 후 자신감 확보 |
| 섹터 간 연쇄 영향을 파악하기 어려움 | 인과관계 그래프로 "금리 인상 → 건설 하락" 시각화 |
| 정보를 찾는 데 시간이 너무 많이 걸림 | AI 코치에게 음성으로 물으면 30초 내 답변 |

### SO 전략 (강점-기회)
- 50대 이상 증가 트렌드에 음성 AI 코치를 결합하여 시니어 전용 투자 도우미 포지셔닝
- 배당 투자 트렌드에 맞춰 배당 시뮬레이터를 핵심 훅(hook) 기능으로 강조

### WT 전략 (약점-위협)
- 증권사와 직접 경쟁 회피: "투자 자문"이 아닌 "투자 학습/정보" 포지셔닝으로 규제 리스크 최소화
- 토스/카카오 대비 차별화: 대형 플랫폼이 제공하지 않는 "음성 AI + 배당 특화" 니치 공략

**음성 UX 플로우 (상세)**:
```
1. 사용자가 마이크 버튼 탭 (또는 "핀이야" 웨이크워드)
2. Whisper STT가 음성 → 텍스트 변환 (< 2초)
3. 변환된 텍스트를 채팅 버블에 표시 ("이렇게 물어볼까요?")
4. 사용자 확인 (또는 자동 전송 설정 가능)
5. Gemini가 포트폴리오/뉴스 컨텍스트와 함께 답변 생성
6. 텍스트 답변을 채팅 버블에 표시 + 동시에 Clova TTS 재생
7. 답변 끝에 "더 궁금한 거 있으세요?" 프롬프트
8. 사용자가 이어서 음성/텍스트로 후속 질문 가능
```

**시니어 모드 UX**:
- 글씨 크기: 기본 18px → 시니어 22px
- 마이크 버튼: 하단 고정, 직경 64px (일반 40px)
- 핀이 캐릭터: 친근한 표정, 답변 중 애니메이션
- 음성 자동 재생: ON (시니어 모드 기본값)
- 복잡한 수치 → "쉬운 말 모드" 자동 활성화
